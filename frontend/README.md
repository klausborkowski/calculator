# Calculator Frontend

React приложение для калькулятора упаковок.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Сборка

```bash
npm run build
```

## Функциональность

- Добавление пакетов через API
- Удаление пакетов через API
- Расчет необходимых пакетов для заказа
- Отображение результатов в таблице
- Интеграция со Swagger UI
